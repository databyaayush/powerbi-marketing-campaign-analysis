# 📊 Marketing Campaign Analysis Dashboard (Power BI + SQL)

## 📌 Project Overview
This project focuses on analyzing marketing campaign performance across multiple channels, regions, and campaign types to evaluate profitability and optimize business decisions.

The dashboard provides insights into campaign ROI, revenue generation, and spending efficiency.

---

## 🎯 Business Problem
Organizations invest heavily in marketing campaigns but often lack clarity on:
- Which campaigns generate the highest ROI
- Which regions perform best
- How spending impacts revenue

This project aims to solve these challenges using data-driven insights.

---

## 🛠 Tools & Technologies
- Power BI (Dashboard & Visualization)
- SQL (Data Analysis)
- Excel/CSV (Dataset)

---

## 📂 Dataset Description
The project uses multiple datasets:
- Marketing Campaign Details
- Campaign Performance Data
- Region Performance Data

---

## 📊 Key Metrics
- Total Revenue: 42.54M
- Total Spend: 25.69M
- Average ROI: 0.68
- Campaign Performance by Type & Region

---

## 📈 Dashboard Features

### 1. KPI Overview
- Total Revenue
- Total Spend
- Average ROI
- Best Performing Campaign

### 2. Regional Analysis
- Revenue vs Spend by Region
- Region-wise performance comparison

### 3. Campaign Analysis
- ROI by Campaign Name
- Spend vs ROI per Campaign
- Top Performing Campaigns

### 4. Campaign Type Insights
- Digital vs Traditional Campaign Distribution

---

## 🔍 Key Insights

- Influencer Marketing generated the highest ROI among all campaigns
- Digital campaigns dominate with ~62% share
- Regions like Asia and North America show strong revenue performance
- Some campaigns show high spend but low ROI → optimization needed

---

## 📌 Conclusion

The analysis highlights the importance of:
- Allocating budget to high ROI campaigns
- Reducing spend on underperforming campaigns
- Focusing on high-performing regions
